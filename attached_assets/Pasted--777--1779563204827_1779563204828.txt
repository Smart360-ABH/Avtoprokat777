Создай для сайта «Автопрокат 777» уникальный интерактивный блок выбора автомобилей в стиле гаража из Need for Speed Underground 2 / Forza Horizon / World of Tanks.

Главная идея:
Пользователь попадает в красивый 3D/псевдо-3D гараж.
Гараж остается статичным, а автомобили плавно меняются при свайпе или нажатии стрелок влево/вправо.

Нужно создать эффект:
- будто машина стоит внутри одного и того же гаража
- пользователь листает автомобили как в игре
- плавная анимация смены авто
- ощущение премиального автосалона/игрового меню

Визуальный стиль:
- современный premium garage
- мягкое освещение
- отражения пола
- неоновые акценты
- cinematic atmosphere
- реалистичные тени
- анимации камеры
- дорогой игровой интерфейс

Функциональность:
- свайп влево/вправо на мобильных
- стрелки переключения на ПК
- плавный transition между машинами
- fade + slide animation
- поддержка touch gestures
- возможность бесконечного перелистывания автомобилей
- lazy loading изображений

Структура гаража:
- фон гаража остается неизменным
- меняется только автомобиль и информация о нем
- автомобиль должен стоять по центру
- при смене авто камера слегка двигается
- сделать ощущение живого интерфейса

Добавить категории автомобилей:
- Эконом
- Бизнес
- Кроссоверы
- Кабриолеты
- Минивэны
- Спорт

Категории должны переключаться:
- красивыми tabs-кнопками
- либо как меню в игре
- при смене категории список автомобилей обновляется

Для каждой машины отображать:
- название
- цена в сутки
- коробка передач
- тип топлива
- количество мест
- краткое описание
- кнопку «Забронировать»
- кнопку «Подробнее»

Добавить эффекты:
- подсветка активного автомобиля
- лёгкое вращение машины при наведении
- glow-эффекты
- motion blur при переключении
- плавное появление текста
- анимированная подсветка пола

Интерфейс:
Сделай UI как в игровой автомобильной галерее:
- минималистично
- дорого
- без перегруза
- крупные элементы
- красивые шрифты
- стеклянные панели (glassmorphism)
- premium buttons

Техническая реализация:
- React + Next.js
- TailwindCSS
- Framer Motion для анимаций
- Embla Carousel или Swiper для свайпов
- оптимизировано для мобильных устройств
- плавные 60fps анимации
- адаптивность

Дополнительно:
- подготовить структуру под будущие 3D модели
- пока использовать качественные PNG/WebP изображения автомобилей
- предусмотреть возможность перехода на Three.js позже
- сделать архитектуру компонентов модульной

Создай компоненты:
- GarageScene
- CarCarousel
- CarCard
- CategoryTabs
- GarageControls
- BookingModal

Добавь демо-автомобили:
- Toyota Soarer
- Mercedes-Benz SLK 320
- Toyota Alphard
- Toyota Mark X Premium
- Honda Fit RS

Главный акцент:
Пользователь должен почувствовать себя как в меню выбора машины в видеоигре.

Сайт должен выглядеть не как обычный автопрокат, а как современный premium automotive experience.
